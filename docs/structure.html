Documentation for the Demand Project

Chapter 1 The structure of the codes

In this chapter, we list the structure of the code for the project. Codes are categorized in the following categories by their usage: 1. data creation 2. analysis 1. data cleaning/manipulation 2. data merge 3. estimations 4. counterfactual analysis 3. table and figure knitting 1. summary statistics 2. regressions 3. histograms and other figures

The following sections display the tree diagrams of the scripts, and show the dependencies of each script. Note that a .R file indicates a single script, and some features may require more than one script to realize.
